Knockout bracket

The elimination rounds are shown from the Round of 32 through the Final. Teams remain placeholders until the group stage locks, but match times and venues are filled in PDT.

Group-stage fixture bracket

All 72 group-stage games, grouped by group, with flags, PDT kickoff times, venues, scores, and points gained when a result is final.

Group standings

Calculated from the final scores currently in the tracker. Each result gives 3 points for a win and 1 for a draw.

All matches by date · PDT

Every World Cup game is grouped by calendar date, then sorted by kickoff time. Search or filter by stage, group, or date.

Daily 10:30 PM PDT update

This page can refresh live data while open. For an actual daily published update, use the included workflow or a local cron job.

Browser mode Keep this page open and it will try the public API shortly after 10:30 PM PDT each day, then store the latest payload in localStorage. The hero snapshot then rolls forward to the next PDT matchday and shows every match scheduled on that date.
GitHub Pages mode Put these files in a GitHub repo, enable Pages, and add the included .github/workflows/update-bracket.yml. The cron is set to 30 5 * * *, which is 10:30 PM PDT during the tournament.
Local server mode Run python update_data.py from this folder, or schedule it with 30 22 * * * cd /path/to/worldcup_bracket_package && python3 update_data.py.
